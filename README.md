拼图游戏 (Puzzle Game)
一个基于 Java Swing 开发的桌面拼图游戏，支持用户登录/注册、多主题图片拼图。

## 🎮 项目简介
- 这是一个课程设计/练手项目，实现了经典拼图游戏的核心玩法
- 包含登录、注册、游戏主界面、多图片主题等完整模块
- 适合 Java GUI 开发初学者学习参考(注册和登陆模块使用文件存储，未来修改成数据库存储)
- 目前有些模块未完成：按下w健后直接胜利之后，按下更换图片成其他图片，也是出现直接胜利的画面，不会重新打乱，重新玩，需要点击重新开始。
- 还有第一张图片路径可以完善，即每次重新登陆都是不同的照片，而不是固定的某一张。
- 还有可以设定某一用户登陆出现美女照片，其他用户登陆为其他女性照片。
